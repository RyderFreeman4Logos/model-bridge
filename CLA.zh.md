# 贡献者许可协议（CLA）- 中文版

## 序言与目的

感谢您对本项目的关注和参与。本协议规定了您向 model-bridge 项目贡献数据的相关条款。

本项目致力于：
- **推进信息自由** — 开发技术工具以对抗专制审查制度
- **保护言论权利** — 建立防止权威主义语言审查的机制
- **维护公共利益** — 为全球追求自由的群体提供技术支持

## 定义

在本协议中：

- **"贡献者"** 指通过标注系统、提交数据或以其他方式向本项目提供数据的个人或组织
- **"贡献数据"** 指通过本项目的标注系统、数据收集工具或其他官方渠道提交的任何原始数据、标注结果或衍生数据
- **"项目"** 指 model-bridge 项目及其所有相关文档、代码、数据和衍生作品
- **"我们/项目方"** 指 model-bridge 项目的维护者和运营组织

## 数据授权许可

通过向本项目提交贡献数据，贡献者向项目方授予：

1. **全球、免费、永久、非独占的许可** — 项目方可以使用、复制、修改、衍生、分发贡献数据
2. **商业和非商业用途** — 包括但不限于：
   - 训练和优化机器学习模型
   - 进行学术研究和技术分析
   - 开发和改进防审查技术工具
   - 支持公益组织和个人
3. **再授权权** — 项目方可以将这些权利授予第三方合作伙伴、研究机构或其他公益组织

## 数据使用说明

### 公益用途承诺

贡献者理解并同意：

- 其贡献数据将**仅用于公益目的**，包括但不限于：
  - 对抗权威主义审查制度
  - 保护言论和信息自由
  - 支持学术研究和技术创新
  - 为受审查压制的群体提供技术帮助

- 项目方**不会将贡献数据用于**：
  - 军事、执法或政府压制用途
  - 违反人权的任何目的

### 不开源说明

贡献者明确知悉并同意：

- **贡献数据不会被公开开源发布** — 虽然项目代码可能以开源形式发布，但贡献数据不会被包含在任何公开可用的数据集中
- **理由说明**：
  - 对投资者的责任 — 项目方因接收了投资资金，需要对投资者的利益负责
  - 公共安全考量 — 防止数据被政府部门或其他恶意行为者获取并用于审查或压制
  - 知识产权保护 — 保护项目方的竞争优势和技术成果
- **访问限制** — 贡献数据仅供项目核心团队和经过严格审查的合作伙伴访问

### 数据治理

- 项目方承诺采取合理的技术和组织措施保护贡献数据的安全
- 项目方不会与第三方共享贡献数据，除非获得贡献者的明确书面同意或出于遵守法律的必要
- 项目方会定期更新数据安全政策并通知贡献者

## 贡献者权利

### 数据下载权

贡献者保留以下权利：

- **可以随时下载自己贡献的数据** — 贡献者可以申请访问和下载其本人通过标注系统提交的所有数据
- **无需理由** — 项目方不会要求贡献者说明下载原因
- **及时提供** — 项目方承诺在收到申请后 30 天内提供数据下载链接
- **原始格式** — 数据将以原始提交的格式或标准的可移植格式提供

### 如何申请下载

贡献者可以通过以下方式申请下载其贡献数据：
- 在项目的 GitHub 仓库提交 Issue，标题为 "Request for Personal Data Download"
- 发送电子邮件至项目维护者，注明贡献者 GitHub 账号

### 撤回贡献

- 贡献者可以申请撤回其贡献数据
- 撤回会在项目方收到申请后的合理时间内执行
- 已使用贡献数据训练的模型的现有副本不需要重新训练或删除

## 陈述与保证

贡献者陈述并保证：

1. **法律权利** — 贡献者拥有或控制贡献数据的所有必要权利，有权授予本协议中所述的许可
2. **第三方权利** — 贡献数据不会侵犯任何第三方的著作权、商标权、专利权或其他知识产权
3. **合法来源** — 贡献数据的收集方式符合适用法律，不包含非法获取的内容
4. **个人数据** — 如果贡献数据包含第三方的个人信息，贡献者已获得必要的同意或拥有法律权利进行处理
5. **真实性** — 贡献数据是真实、准确和完整的，不包含虚假或误导性内容

## 免责声明

1. **"按原样"提供** — 贡献数据按"现状"提供，项目方不提供任何形式的保证，包括但不限于适销性、特定用途适用性或不侵权的隐含保证

2. **不保证准确性** — 项目方不保证贡献数据的准确性、完整性或及时性

3. **免责限制** — 在适用法律允许的最大范围内，项目方及其贡献者、合作伙伴和投资者对因使用贡献数据而产生的任何直接、间接、特殊、偶发或后果性损害赔偿不承担任何责任

4. **法律合规** — 贡献者应确保其贡献数据的提交符合其所在司法管辖区的所有适用法律和规定

## 法律条款

### 一般条款

1. **完整协议** — 本协议构成贡献者与项目方之间关于贡献数据的完整协议，取代所有先前的协议或理解

2. **修改** — 项目方可能不时修改本协议。如果进行实质性修改，项目方会通过在 GitHub 仓库发布通知或向贡献者发送电子邮件来告知

3. **准据法** — 本协议的解释和执行遵循适用法律

4. **异议解决** — 任何关于本协议的异议应首先通过友好协商解决

### 知识产权

- 贡献数据的知识产权所有权保持不变
- 贡献者保留对其贡献数据的所有道德权利（如适用）
- 项目方可以添加著作权声明，但必须承认贡献者

### 终止

- 本协议自贡献者首次提交数据时生效，无限期持续
- 贡献者可以通过停止提交数据来有效退出新协议的适用
- 已授予的许可不会因协议终止而终止

## 签署

通过向 model-bridge 项目提交贡献数据，贡献者同意受本协议约束。

### 电子签署

贡献者通过以下方式在电子形式上签署本协议：

1. **GitHub 账号** — 使用您的 GitHub 账号提交数据时，您同意受本协议约束
2. **拉取请求评论** — 通过在 CLA 相关的拉取请求中留言同意本协议条款
3. **数据提交** — 通过项目的标注系统提交数据时，您同意受本协议约束
